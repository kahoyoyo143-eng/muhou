# 幕後 — 个人创作操作系统 · 项目概览

## 这是什么

"幕後"是一个纯前端的单 HTML 文件 Web App，定位为**个人创作操作系统**。核心理念：你不是在"过生活"，你是在"制片"你的人生。

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | 单个 HTML 文件 | 内含所有 CSS + JS，约 2155 行 |
| 数据库 | Supabase (PostgreSQL) | 云端数据库，免费额度 |
| AI | DeepSeek API | 还没接 |
| 存储 | 浏览器本地 | 图片 base64 存 IndexedDB |
| 部署 | 无需部署 | 双击 HTML 即用 |
