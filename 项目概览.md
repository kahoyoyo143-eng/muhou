# 幕後 — 个人创作操作系统 · 项目概览

## 这是什么

"幕後"是一个纯前端的单 HTML 文件 Web App，定位为**个人创作操作系统**。核心理念：你不是在"过生活"，你是在"制片"你的人生。

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | 单个 HTML 文件 | 内含所有 CSS + JS，约 5800 行 |
| 数据库 | Supabase (PostgreSQL) | 云端数据库，免费额度 |
| AI | DeepSeek API | 已接入，AI 助手可用 |
| 存储 | 浏览器本地 + Supabase | localStorage（座右铭），Supabase（所有业务数据） |
| 部署 | GitHub Pages | legacy 模式，main 分支根目录直接部署 |

## 已完成的模块

| 模块 | 状态 | 说明 |
|------|------|------|
| 🏠 OKR+习惯 | ✅ 可用 | OKR 瀑布流双栏 + 四级（O→KR→任务→子任务）+ 归档 + 习惯月历打卡 + 热力图 |
| 📌 今日3件事 | ✅ 可用 | 从 OKR 挑选，含自定义事项，持久保留 |
| 📝 备忘录 | ✅ 可用 | 今日3件事旁并排，可打勾，完成计入复盘统计 |
| 🌟 愿景版 | ✅ 可用 | 软木板便签拼贴，文字+图片，N年/1年两个Tab |
| 📥 灵感库 | ✅ 可用 | 卡片网格、状态筛选、标签云、模糊搜索、图片上传 |
| 📔 日记 | ✅ 可用 | 月历浏览、心情选择、自动记录时间线、自由书写、图片上传 |
| 📋 月度复盘 | ✅ 可用 | 数据自动汇总 + 5步引导式复盘（含💰财务回顾） |
| 💰 财务回顾 | ✅ 可用 | 收入/支出/结余/储蓄 + 自由书写，每月复盘时填写 |
| 💯 100件小事 | ✅ 可用 | 看板式年度清单，分类可编辑+160+Emoji+拖拽排序+自定义完成时间+单条删除+复盘统计 |
| ⚡🎲 捕捉·回顾 | ✅ 可用 | 左：快速捕捉 + 最近列表；右：塔罗式随机抽卡回顾 |
| 🤖 AI助手 | ✅ 可用 | 5 个快捷分析 + 自由提问，接 DeepSeek API |
| 💬 座右铭 | ✅ 可用 | 可编辑激励语，显示在OKR页面 |
| ⚙️ 设置 | ✅ 可用 | Supabase 配置、DeepSeek Key 输入 |
