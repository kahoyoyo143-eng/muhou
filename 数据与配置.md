## 数据库表结构

### `okrs` — OKR 目标

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid (自动生成) | 主键 |
| title | text | 目标名称，例如「成为有个人风格的短片导演」 |
| period_q | text | 所属季度，如 `2026-Q2` |
| period_m | text | 所属月份（可选），如 `2026-06` |
| deadline | text | 截止日期，格式 `YYYY-MM-DD` |
| memo | text | 备忘录/备注 |
| key_results | jsonb | KR 数组，结构见下方 |
| created_at | timestamptz | 创建时间 |

### `habits` — 习惯

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| title | text | 习惯名称 |
| emoji | text | 图标 emoji |
| rhythm | text | `daily` 或 `weekly` |
| target_count | int | 每周目标次数 |
| weekly_days | int[] | 每周哪几天（如 [1,3,5]） |
| rest_days | int[] | 休息日（0=周日） |
| checkins | jsonb | 打卡记录 `{"2026-06-15": true}` |
| archived | bool | 是否归档 |
| created_at | timestamptz | 创建时间 |

### `daily_items` — 每日3件事

| 字段 | 类型 | 说明 |
|------|------|------|
| date | text (主键) | 日期，如 `2026-06-22` |
| items | jsonb | 事项数组，结构见下方 |

### `diaries` — 日记

| 字段 | 类型 | 说明 |
|------|------|------|
| date | text (主键) | 日期 |
| content | text | 日记内容 |
| mood | int | 心情 1-5 |
| photos | jsonb | 照片数组 |
| auto_entries | jsonb | 自动记录（OKR完成/习惯打卡） |

### `visions` — 愿景版

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| scope | text | `n-year` 或 `1-year` |
| cards | jsonb | 便签卡片数组 |

## 关键数据结构（JSONB）

### key_results（OKR 下的 KR）

```json
[
  {
    "title": "KR1: 完成3部短片",
    "tasks": [
      {
        "title": "写剧本初稿",
        "done": false,
        "doneAt": null,
        "subtasks": [
          { "title": "确定主题", "done": false },
          { "title": "写大纲", "done": false }
        ]
      }
    ]
  }
]
```

### daily_items.items（今日3件事）

```json
[
  {
    "title": "写剧本初稿",
    "done": false,
    "source": "okr",
    "objTitle": "成为有个人风格的短片导演",
    "taskRef": { "objId": "xxx", "krIdx": 0, "taskIdx": 0 }
  },
  {
    "title": "回复合作邮件",
    "done": true,
    "source": "custom"
  }
]
```

### habits.checkins（打卡记录）

```json
{
  "2026-06-15": true,
  "2026-06-16": true,
  "2026-06-18": true
}
```

## 重要全局变量（JS）

| 变量 | 说明 |
|------|------|
| `sb` | Supabase 客户端实例 |
| `_okrList` | 所有 OKR 数据（内存缓存） |
| `_habitList` | 所有习惯数据 |
| `_todayItems` | 今日3件事数组 |
| `_editingOKRId` | 编辑中的 OKR ID，null=新建 |
| `_habitCalMonth/_habitCalYear` | 习惯月历当前查看的年月 |
| `window._pickTasks` | 从 pickTodayItems 传给 confirmPickToday 的任务列表 |
